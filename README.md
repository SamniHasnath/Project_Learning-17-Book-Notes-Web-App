# 📚 Book Notes Web App

A modern Full-Stack application designed to help users manage their personal library, store insightful notes, and track reading progress. This project bridges the gap between a clean user experience and a robust relational database.

---

## 🚀 Features

* **Comprehensive Book Management:** Add books with titles, authors, ISBNs, and personal ratings (1–10 scale).
* **Persistent Notes:** Store and edit detailed reading notes for every book in your collection.
* **Dynamic Data Fetching:** Integrates with the **Open Library API** to automatically pull book covers and metadata.
* **Smart Organization:** Sort your library by the newest additions or highest ratings.
* **Responsive UI:** A mobile-first, dashboard-style design featuring **Dark Mode** support.

---

## 🛠️ Tech Stack

| Layer | Technology |
| :--- | :--- |
| **Frontend** | EJS (Embedded JavaScript Templates), Custom CSS |
| **Backend** | Node.js, Express.js |
| **Database** | PostgreSQL |
| **API** | Open Library API |
| **Environment** | Dotenv, PG (Node-Postgres) |

## 💡 Learning Purpose

This project served as a deep dive into the **PERN stack** philosophy, focusing on:
* **Advanced CRUD:** Managing data relationships between book metadata and user notes.
* **SQL Mastery:** Writing optimized PostgreSQL queries for sorting and filtering.
* **API Integration:** Handling asynchronous API calls and external data flow.
